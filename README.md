# My Pomodoro

A customizable desktop **Pomodoro Timer** built with **C# WinForms** to practice UI controls, event-driven programming, and desktop application development.

<img width="1536" height="1024" alt="MyPomodoroThumbnail" src="https://github.com/user-attachments/assets/70da30e1-85dc-46a3-a639-c401d4004cef" />

## Features

- Set custom **Focus** and **Rest** durations
- Start any session freely without fixed order
- **Start / Pause / Reset** controls
- Choose a custom alarm sound or use the default built-in sound
- Control notification volume
- Tray notification using **NotifyIcon** when time is up
- Click notification to dismiss it and reopen the app
- Automatic switch between **Focus** and **Rest** sessions

## Technologies

- C#
- .NET WinForms
- Visual Studio

## Purpose of the Project

This project was built to strengthen practical understanding of WinForms controls and improve problem-solving through real implementation.
